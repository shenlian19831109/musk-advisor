## Tool 2: 10X Thinking Checklist

### Questions to Ask
1. **Cost 10X Reduction**
   - [ ] Can we make it for 1/10 the cost?
   - [ ] What would need to change?
   - [ ] Is it physically possible?

2. **Speed 10X Increase**
   - [ ] Can we do it 10X faster?
   - [ ] What are the real bottlenecks?
   - [ ] Can we parallelize?

3. **Performance 10X Improvement**
   - [ ] Can it be 10X better?
   - [ ] What are the theoretical limits?
   - [ ] Are we optimizing the wrong metric?

4. **Market 10X Expansion**
   - [ ] Can we reach 10X more customers?
   - [ ] What prevents adoption?
   - [ ] Can we change the adoption curve?

### Scoring
- 0-1 checks: Incremental thinking
- 2-3 checks: Moderate ambition
- 4 checks: True 10X thinking

## Tool 4: Musk-Style Decision Tree

### When to Use
Use this for major decisions (hiring, investment, strategy, product).

### Decision Framework

**Level 1: Mission Alignment**
```
Question: Does this align with the ultimate mission?
→ Yes → Continue
→ No → Reject (unless it's a necessary evil with clear exit)
```

**Level 2: First Principles Check**
```
Question: Does this make sense from physics/math?
→ Yes → Continue
→ No → Reject or redesign
```

**Level 3: 10X Potential**
```
Question: Does this have 10X potential?
→ Yes → High priority
→ No → Low priority (unless blocking something else)
```

**Level 4: Talent Density**
```
Question: Do we have/will we get top talent for this?
→ Yes → Proceed
→ No → Fix talent first
```

**Level 5: Failure Analysis**
```
Question: What's the worst that can happen?
→ Acceptable risk → Proceed
→ Unacceptable risk → Redesign or reject
```

### Example: Hiring Decision
```
Mission Alignment: Yes (supports Mars mission)
First Principles: Yes (engineering role needed)
10X Potential: Yes (top engineer = 10X output)
Talent Density: Yes (candidate is top 1%)
Failure Analysis: Acceptable (if fails, can replace)

→ Decision: HIRE
```
