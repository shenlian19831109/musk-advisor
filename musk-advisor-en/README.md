# Musk Advisor (English Version)

Get Musk-style analysis for business, organization, self-improvement, strategy, and AI questions.

## Quick Start

### How to Use

1. Copy this directory to Hermes Agent's skills directory:
```bash
cp -r musk-advisor-en ~/.hermes/skills/productivity/
```

2. Tell Hermes: **"Use Musk's framework to analyze..."**

### Questions You Can Ask

- **Business/Strategy**: "I want to enter XX market, but competition is fierce"
- **Organization/Management**: "Team is getting bigger, efficiency dropping"
- **Self/Career**: "No motivation, don't know what I want"
- **AI/Tech**: "AI is developing too fast, what's your take"
- **Innovation/Startup**: "I have an idea, not sure if it'll work"

## Core Frameworks

### Core Motivation
- Existential anxiety → Civilization backup
- Mission double helix: SpaceX (transport) + Tesla (energy consumption) + SolarCity (energy production)

### Thinking Methods
- **First principles**: Don't accept "industry is like this", only accept "physics/chemistry/math"
- **10X thinking**: Not improve by 10%, but think how to make cost 1/10
- **Backwards planning**: From goal to today's actions

### Execution Patterns
- **Vertical integration**: Control the full chain from raw materials to user
- **Talent density**: Top talent costs 2-3X more, produces 5-10X more
- **Extreme commitment**: 12-18 hours/day, 100-120 hours/week
- **Failure culture**: Fail → Analyze → Learn → Improve

## Blind Spots

Musk has blind spots too, watch out when giving advice:
1. Over-confidence: Underestimates time, overestimates tech maturity
2. Ignores "human" factors: Thinks equity solves all motivation
3. Public image too personalized: Personal IP too strong, company fate tied to individual
