### 4. 马斯克盲点

**5大失误记录：**

| 时间 | 事件 | 后果 | 根因 |
|------|------|------|------|
| 2018年8月 | 发推"考虑以420美元私有化" | SEC起诉，罚款2000万美元，丢董事长 | 情绪化决策 |
| 2017-2018年 | 过度自动化 | Model 3产能危机加剧 | 工程师思维 vs 现实 |
| 2022年 | Twitter大规模裁员 | 品牌受损，广告主流失 | 执行过于激进 |
| 2016年 | SolarCity收购 | 特斯拉股东诉讼 | 关联交易判断失误 |
| 2019年 | 发推"特斯拉股价太高" | 股价当天跌10% | 公开言论失控 |

**3深层盲点：**

```
盲点1：过度自信

表现：
→ 经常低估完成时间（火箭/汽车/隧道）
→ 经常高估技术成熟度（自动驾驶/FSD）

原因：
→ 物理上能实现 ≠ 按时能实现
→ 工程能力 ≠ 管理能力

────────────────────────────────────────────────────────

盲点2：忽视"人"的因素

表现：
→ 觉得员工都应该像他一样拼命
→ 觉得"给股权"就能解决一切激励
→ 裁员后才发现有些人是不可替代的

原因：
→ 动机不同：有人要使命，有人要生活
→ 信息密度不同：他的太高，别人跟不上

────────────────────────────────────────────────────────

盲点3：公众形象过于个人化

表现：
→ 特斯拉股价经常随他一条推特波动
→ 他的个人行为会影响公司品牌
→ 抽大麻/争议言论

原因：
→ 个人IP太强，公司命运绑定个人
→ 好的：省了大量广告费
→ 坏的：风险高度集中
```
