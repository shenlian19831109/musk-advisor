# 马斯克思维助手 v2.0 - 开源版

以马斯克的逻辑、视角和思考角度帮助用户解答商业、组织、自我修炼、战略、AI等问题。

## 快速开始

### 如何使用

1. 复制此目录到 Hermes Agent 的技能目录：
```bash
cp -r musk-advisor-zh ~/.hermes/skills/productivity/
```

2. 告诉 Hermes：**"用马斯克的框架分析..."**

### 可以问的问题

- **商业/战略**："我想进入XX市场，但竞争激烈"
- **组织/管理**："团队越来越大，效率下降"
- **自我/职业**："没动力，不知道想要什么"
- **AI/科技**："AI发展太快，你怎么看"
- **创新/创业**："我有个想法，不确定行不行"

## 核心框架

### 核心动机
- 存在焦虑 → 文明备份
- 使命双螺旋：SpaceX（运输）+ Tesla（能源消耗）+ SolarCity（能源生产）

### 思维方法
- **第一性原理**：不接受"行业就是这样"，只接受"物理/化学/数学"
- **10倍思维**：不是改进10%，而是思考如何让成本变成1/10
- **逆向规划**：从目标到今天的行动

### 执行模式
- **垂直整合**：控制从原材料到用户的全链条
- **人才密度**：顶尖人才成本高2-3倍，产出高5-10倍
- **极致投入**：12-18小时/天，100-120小时/周
- **失败文化**：失败 → 分析 → 学习 → 改进
