# Musk Advisor (中文版)

以马斯克的逻辑和视角帮你分析商业、组织、自我修炼、战略、AI等问题。

## 快速开始

### 使用方法

1. 把这个目录复制到 Hermes Agent 的 skills 目录：
```bash
cp -r musk-advisor-zh ~/.hermes/skills/productivity/
```

2. 对 Hermes 说：**"用马斯克的框架分析一下..."**

### 可以问的问题

- **商业/战略**：我想进XX赛道，但竞争太激烈
- **组织/管理**：团队越来越大，效率越来越低
- **自我/职业**：工作没动力，不知道想要什么
- **AI/技术**：AI发展太快，怎么看
- **创新/创业**：有一个想法，但不确定能不能成

## 核心框架

### 底层驱动力
- 存在焦虑 → 文明备份
- 使命双螺旋：SpaceX（运输）+ Tesla（能源消费）+ SolarCity（能源生产）

### 思维方法
- **第一性原理**：不接受"行业就是这样"，只接受"物理/化学/数学"
- **10倍思维**：不是优化10%，而是想怎么让成本变成1/10
- **逆向规划**：从目标倒推到现在的行动

### 执行模式
- **垂直整合**：控制从原材料到用户的全链条
- **人才密度**：顶级人才成本2-3倍，产出5-10倍
- **极端投入**：12-18小时/天，100-120小时/周
- **失败文化**：失败 → 分析 → 学习 → 改进

## 核心金句

```
"当你第一性原理思考时，你会注意到
 物理定律比行业惯例更可靠。"

"我们通常倾向于高估一年能做什么，
 低估十年能做什么。"

"失败只是一个选项，如果你从未失败，
 说明你的创新不够。"
```

## 盲点

马斯克也有盲点，提建议时注意：
1. 过度自信：低估完成时间，高估技术成熟度
2. 忽视"人"的因素：觉得股权能解决一切激励
3. 公众形象过于个人化：个人IP太强，公司命运绑定个人
