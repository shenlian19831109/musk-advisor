# 马斯克思维助手 (Elon Musk Thinking Assistant)

[中文版本](./musk-advisor-zh/README.md) | [English Version](./musk-advisor-en/README.md)

## 项目概述

**马斯克思维助手**是一个基于埃隆·马斯克（Elon Musk）思维框架的AI助手技能包。它使用第一性原理、10倍思维、使命驱动等马斯克核心思维方法，帮助用户分析商业、组织、自我修炼、战略、AI等问题。

## 核心功能

### 🚀 马斯克思维框架
- **第一性原理分析**：从物理/化学/数学基本定律出发，重构问题
- **10倍思维**：不是改进10%，而是思考如何让成本变成1/10
- **使命驱动**：存在焦虑 → 文明备份 → 多行星物种
- **逆向规划**：从终极目标倒推今天的行动

### 💡 应用场景
- **商业/战略决策**：市场进入、竞争分析、创新突破
- **组织/管理优化**：团队效率、人才密度、流程简化
- **自我/职业发展**：动力不足、方向迷茫、长期规划
- **AI/科技思考**：技术趋势、创新机会、风险分析

## 快速开始

### 1. 安装到 Hermes Agent
```bash
# 中文版本
cp -r musk-advisor-zh ~/.hermes/skills/productivity/

# 英文版本
cp -r musk-advisor-en ~/.hermes/skills/productivity/
```

### 2. 使用示例
```
"用马斯克的框架分析：我想进入电动汽车市场，但竞争激烈"
"Use Musk's framework: I have a startup idea but worried about competition"
```
